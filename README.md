# VPyLab — 반딧불 은하 오르골

VPyLab 문서와 예제를 살펴본 뒤 만든 작은 3D 우주 풍경입니다.
중심 별 주위를 행성이 돌고, 유성이 지나가며, 반딧불이와 오로라 리본이 리듬처럼 움직입니다.

- 실행 페이지: https://thinkervis.github.io/vpylab-thinkervis-test-20260518-2030/
- 주요 기능: `sphere`, `ring`, `helix`, `label`, `local_light`, `attach_trail`, `rate`
- 협업 초대 코드: `cde3639d`
- 항해 일지: [`VOYAGE.md`](./VOYAGE.md)
